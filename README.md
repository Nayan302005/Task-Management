## 📌 About The Project

A simple and efficient **Task Management Web Application** built using **Python and Django**.
This project allows users to **create, view, complete, and delete tasks** through an interactive dashboard interface.

---

## ✨ Features

* ➕ Add new tasks
* 📋 View all tasks on dashboard
* ✅ Mark tasks as completed
* 🗑️ Delete tasks
* ⚙️ Manage tasks through Django Admin Panel
* 📂 Data stored using SQLite database

---

## 🛠️ Tech Stack

* Python
* Django
* HTML
* CSS
* SQLite
* Django Admin

## ⚙️ Run the Project

```bash
git clone https://github.com/yourusername/taskmanager.git
cd taskmanager
pip install django
python manage.py migrate
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
